# InternetReferences Field


Microsoft Outlook email object (.msg) and MIME (.eml) email associated metadata. Contains a list of MIME message IDs that specify the messages to which this reply is related (corresponds to MAPI PidTagInternetReferences property). "References:" MIME header is also mapped to this metadata field.
