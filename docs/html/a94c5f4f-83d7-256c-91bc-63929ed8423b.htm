Remarks

The helper lists of metadata fields in this class can be used to populate UI drop down lists of fields to search for in full-text search indexes, for example.

This class uses reflection to create lists of the defined fields and field attributes in classes KnownBooleanMetadataFields, KnownDateTimeMetadataFields, KnownDoubleMetadataFields, KnownInt32MetadataFields, KnownInt64MetadataFields, and KnownStringMetadataFields.
