System and Development Requirements

  Important

The .NET 8 assemblies that comprise the Open Discover SDK for .NET are x64 release builds (not AnyCPU) due to x64 dependencies. Therefore, applications that reference and use the SDK assemblies must also be x64 builds.

System Requirements:
  • .NET version - .NET 8 required
  • Minimum supported client - Windows 7 (64-bit x64)
  • Minimum supported server - Windows Server 2008 (64-bit x64)

Supported .NET :

  • .NET 8
