      Supported File Formats

      The following sections below document Open Discover® SDK for .NET supported file formats for identification and the level of content extraction supported (if any). The supported file formats are grouped by file format classification (see IdClassification) and then by each file format identification (see Id) that belongs to that format classification.

        Note

      The .NET assemblies that make up Open Discover SDK are x64 release builds (not AnyCPU) due to x64 dependencies. Therefore, applications that reference and use the SDK assemblies MUST also be x64 builds.
