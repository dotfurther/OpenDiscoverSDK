## Remarks

This interface allows for extracting text via a proprietary binary-to-text algorithm. The algorithm supports binary-to-text for UTF-16 (latin Unicode range), UTF-8, and code page 1252 encodings.

"Large" is a subjective term defined by the <a href="90aee97f-a132-9d0b-5c91-d6ac2eb95ace">LargeDocumentCritera</a> property value.
